Basic Structure Elements – Declares the type of document and version of HTML. – Root element that contains all the HTML content. – Contains metadata, title, styles, and links to scripts or stylesheets. – Sets the title of the web page (appears in the browser tab). <body> – Holds all the visible content of a web page. Text Content Elements <h1> to <h6> – Headings, with <h1> being the largest and <h6> the smallest. <p> – Paragraph of text. <span> – Inline container for styling or grouping small pieces of text. <br> – Line break. <hr> – Horizontal line (thematic break). Formatting Elements <b> / <strong> – Bold text (<strong> also indicates importance). <i> / <em> – Italic text (<em> emphasizes meaning). <u> – Underlined text. <mark> – Highlighted text. <small> – Smaller text. <sup> – Superscript (e.g., x²). <sub> – Subscript (e.g., H₂O). Links and Media <a> – Anchor tag, used for hyperlinks. <img> – Embeds images. <audio> – Embeds audio files. <video> – Embeds video files. <iframe> – Embeds another webpage within the current page.
