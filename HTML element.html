Basic Structure Elements – Declares the type of document and version of HTML. – Root element that contains all the HTML content. – Contains metadata, title, styles, and links to scripts or stylesheets. – Sets the title of the web page (appears in the browser tab). <body> – Holds all the visible content of a web page. Text Content Elements <h1> to <h6> – Headings, with <h1> being the largest and <h6> the smallest. <p> – Paragraph of text. <span> – Inline container for styling or grouping small pieces of text. <br> – Line break. <hr> – Horizontal line (thematic break). Formatting Elements <b> / <strong> – Bold text (<strong> also indicates importance). <i> / <em> – Italic text (<em> emphasizes meaning). <u> – Underlined text. <mark> – Highlighted text. <small> – Smaller text. <sup> – Superscript (e.g., x²). <sub> – Subscript (e.g., H₂O). Links and Media <a> – Anchor tag, used for hyperlinks. <img> – Embeds images. <audio> – Embeds audio files. <video> – Embeds video files. <iframe> – Embeds another webpage within the current page. Lists <ul> – Unordered list (bullet points). <ol> – Ordered list (numbered). <li> – List item inside <ul> or <ol>. <dl> – Description list. <dt> – Term in a description list. <dd> – Description of the term in a description list. Tables <table> – Creates a table. <tr> – Table row. <td> – Table data cell. <th> – Table header cell. <thead> – Groups header content. <tbody> – Groups body content. <tfoot> – Groups footer content. <caption> – Table title. Forms <form> – Collects user input. <input> – Input field (text, number, checkbox, etc.). <textarea> – Multi-line text input. <button> – Clickable button. <select> – Dropdown menu. <option> – Option in a dropdown menu. <label> – Label for form elements. <fieldset> – Groups related form fields. <legend> – Title for a <fieldset>. Semantic Elements <header> – Introductory content or navigation for a page/section. <nav> – Navigation links. <main> – Main content of the document. <section> – Defines a thematic grouping of content. <article> – Self-contained piece of content (e.g., blog post). <aside> – Sidebar content, often supplementary. <footer> – Footer content for a page/section. <figure> – Groups images/media with caption. <figcaption> – Caption for a <figure>.
